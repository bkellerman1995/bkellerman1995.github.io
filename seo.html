¿Qué es SEO?

SEO significa Search Engine Optimization (Optimización para motores de búsqueda). Se trata del conjunto de estrategias y técnicas de optimización que se hacen en una página web para que aparezca orgánicamente en buscadores de Internet como Google, Yahoo o Youtube.

¿Cómo optimizar una web para SEO?

  1. Darse de alta en Search Console.
  2. Hacer un pequeño estudio de palabras clave.
  3. Crear una buena estructura de urls que contengan la palabra clave.
  4. Tener los títulos optimizados.
  5. Redactar descripciones con la palabra clave con texto atractivo para convertir.
  6. Optimizar las imágenes (nombre del fichero, título alt, tamaño en píxeles y peso (B,MB).

¿Qué son las metaetiquetas en HTML?

Son etiquetas utilizadas para describir metadatos (información sobre datos). Los metadatos no se mostrarán en la página, pero serán analizables por máquina.

¿Para qué sirven las metaetiquetas (metadatos)?

Se utilizan típicamente para especificar la descripción de la página, las palabras clave, el autor del documento, la última modificación y otros metadatos.

Los metadatos pueden ser utilizados por los navegadores (cómo mostrar el contenido o la página de recarga), los motores de búsqueda (palabras clave) u otros servicios web.

Estructura de la etiqueta meta

Ejemplos de etiquetas Meta dentro de la cabecera HTML (<head>)
Nombre Descripción Sintaxis
UTF-8 Es el responsable de que el navegador o el cliente de correo muestre el contenido del texto correctamente decodificado, sin errores ni caracteres extraños. <meta charset="UTF-8">
description Sirve para describir brevemente el contenido de la página web. Esta información se muestra como snippet (una síntesis en dos líneas del tema de una página que aparece bajo la URL) en los buscadores de uso más generalizado como Google o Bing, por lo que se recomienda cuidar su redacción. <meta description content="Web">
keywords Sirven como criterios a los que responde un buscador para ofrecerle al usuario páginas HTML como respuesta, donde tales palabras clave son parte de los meta tags. <meta keywords content="HTML,CSS,XML,JavaScript"">
author Sirve para hacer referencia al diseñador de una página web y al propietario de los derechos del código fuente de una página HTML. <meta author content="John Doe"">
viewport Sirve para que el sitio web pueda verse y funcionar de manera óptima en dispositivos móviles <meta viewport content="width=device-width, initial-scale=1.0""">

Relación entre MetaEtiquetas y SEO

Las etiquetas meta en HTML juegan un papel importante en SEO (Search Engine Optimization), ya que proporcionan información a los motores de búsqueda sobre el contenido de una página web. Estas etiquetas no son visibles para los usuarios, pero ayudan a los motores a entender mejor la página, lo que puede influir en cómo se clasifica en los resultados de búsqueda.
